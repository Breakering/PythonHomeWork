### (5)本次作业实现的需求：

### (6)测试：

1) 直接打开index.html即可进入购物商城页面，点击登录可进入登录页面，点击成为会员可进入注册页面

2) 直接打开login.html即可进入登录页面，点击免费注册可登录注册页面，点击登录可进入后台管理页面

3) 直接打开register.html即可进入注册页面，点击登录可进入登录页面

4) 直接打开admin.html即可进入后台管理页面
